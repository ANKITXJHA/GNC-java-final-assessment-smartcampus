## Problem
Build a Smart Campus Management System in Java.

## Features
- Add Students and Courses

- View all Students and Enrollments
- Multithreading for Enrollment Processing
- Custom Exception Handling
- File Saving (Bonus)

## How to Run
javac AN_SmartCampus.java
java AN_SmartCampus

## Output
Menu-driven console application with full OOP implementation.
